\subsection{Jupyter Lab}

To Jupyter Lab αποτελεί μια εφαρμογή για την συγραφή και εκτέλεση computational notebook 
(Υπολογιστικών σημειωματαρίων) της Project Jupyter .
Ένα computational notebook της Jupyter είναι ένα έγγραφο που μπορεί να κοινοποιηθεί και το οποίο συνδυάζει κώδικα υπολογιστή, 
περιγραφές σε απλή γλώσσα, δεδομένα, εμπλουτισμένες οπτικοποιήσεις όπως τρισδιάστατα μοντέλα, διαγράμματα, γραφήματα και σχήματα, 
καθώς και διαδραστικά στοιχεία ελέγχου \cite{jupyterlab}.

Οι εφαρμογές τις project Jupyter για την δημιουργία και εκτέλεση των notebook της Jupyter 
μπορούν να χρησιμοποιήσουν πολλές διαφορετικές γλώσσες.
Για την python που θα χρησιμοποιήσουμε,η εκτέλεση θα γίνεται μέσω της IPython kernel,που αποτελεί και το βασικά kernel
των εφαρμογών Project Jupyter.

\subsection{Xαρακτηριστικές βιβλιοθήκες}

Οι βιβλιοθήκες εγκαταστάθηκαν χάρη στην χρήστη του pip,που αποτελεί τον βασικό διαχειριστής πακέτων της python \cite{pipPython}.
H pip είναι και αυτή που κατεβάζει και εγκαθιστά το jupyterLab στον υπολογιστή.

Οι εντολές εγκατάστασεις βρίσκονται εκτός των notebooks,οπότε θα πρέπει να κατέβουν και να εγκατασταθούν ξεχωριστά 
με τις ανάλογες  \texttt{pip install} εντολές,για τον χρήστη που θελήσει να εκτελέσει τα notebook της ανάλυσης δεδομένων.
Η pip μπορεί να εγκαταστήσει πακέτα με τρόπο που είτε θα είναι καθολικά τα πακέτα,είτε θα είναι σε κάποιο ξεχωριστό,virtual περιβάλλον.
Χρησιμοποιήθηκαν και οι δύο τρόποι,σε διαφορετικούς υπολογιστές.

Παρακάτω θα αναφέρουμε κάποιες βιβλιοθήκες που τέλεσαν νευραλγικό ρόλο στο έργο.
Όλες οι παρακάτων βιβλιοθήκες αποτελούν πολύ δημοφιλοί και χρησιμοποιούμενες βιβλιοθήκες για την ανάλυση δεδομένων στην python.
Δεν θα αναφερθούμε σε κάποια βιβλιοθήκη που ανοίκει είδη στις τυπικές βιβλιοθήκες της Python (The Python Standard Library),
που εγκαθιστούνται στον υπολογιστή μαζί με την εγκατάσταση της γλώσσας Python.

\paragraph{requests}
H Requests την χρησιμοποιούμε για την απλοποίηση χρήσης http αιτήσεων.
Την χρησιμοποιούμε για την επικοινωνία του notebook με τον server \cite{requests}

\paragraph{numpy}
Η Numpy αποτελεί την βασική βιβλιοθήκη για τον επιστημονικό υπολογισμό στην Python,
με όλες τις παρακάτω βιβλιοθήκες να την χρησιμοποιούν .
Αυτό συμβαίνει γιατί η δομή δεδομένων που προσφέρει,το \texttt{ndarray},
που είναι πολυσιάστατος πίνακας ομείων σε τύπο αντικείμενο, έχει υλοποιηθεί με τέτοιο
τρόπο από την numpy,ώστε να προσφέρει μεγάλη ταχύτητα στους υπολογισμούς μεταξύ των πινάκων
και μεγάλη ευελιξία \cite{numpy}.

\paragraph{pandas}

Από την Pandas αντλούμε την κύρια δομή δεδομένων 
με την οποία αναλύουμε και επεξαργαζόμαστε τα δεδομένα,το \texttt{Dataframe},
με το \texttt{Series} να είναι η μονοδιάσταση εκδοχή των Dataframes \cite{pandas}.
Χειρίζεται πολύ εύκολα ετερογενής δεδομένα, μέσω της μορφής πίνακα, και χρονοσειρές.

\paragraph{matplotlib και seaborn}

Η matplotlib είναι η κύρια βιβλιοθήκη της python για την δημιουργία γραφημάτων για την οπτιοκοποίηση των δεδομένων,
με μία πληθώρα επιλογών ,όπως γραφήματα διασποράς (scatter plots),ραβδογράμματα (bar plots) και άλλα \cite{matplotlib}.
Η seaborn βασίζεται πάνω στην matplotlib,για ομορφότερη οπτικοποίηση και ευκολότερη δημιουργία γραφημάτων
για τις δομές δεδομένων της pandas \cite{seaborn}. 

\paragraph{scipy}

Η scipy αποτελείται από ένα σύνολο συναρτήσεων,διάφορων μαθηματικών αλγορίθμων και συναρτήσεων διευκόλυνσης,
για πολλά διαφορετικά πανεπιστημιακά πεδία  \cite{scipy}.
Την χρησιμοποιούμε για την υλοποίηση των \texttt{least\_squares} και \texttt{savgol\_filter}.

\paragraph{Scikit-Learn}

H Scikit-Learn,sklearn στο όνομα πακέτου, υλοποιήει μία μεγάλη γκάμα παραδοσιακών μηχανικών μοντέλων,με πολύ εύκολο και προσιτό τρόπο\cite{scikitLearn}.
Δεν χρησιμοποιούμε βιβλιοθήκες για βαθιά μηχανική μάθηση,όπως η  Pytorch και η Tensorflow,για λόγους που θα αναλύσουμε στο επόμενο κεφάλαιο.
